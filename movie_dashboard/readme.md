## Installation

- Clone repository
- Run <code>composer install</code>
- Rename .env.example to .env
- Run <code>php artisan key:generate</code>
- Run <code>npm install</code>
- Update your Bootstrap version to Bootstrap 4.1
- Run <code>npm run watch</code> to compile assets
